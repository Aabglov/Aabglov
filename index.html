How

I use a lot of different tools and skills in my day-to-day. The first things I use everyday are an 8oz single-shot Mocha and a Mac. I'll happily program on a Linux box as well. Or a tablet connected to an AWS instance. I've even programmed on my phone once. Basically anything other than a Windows machine.

My IDE of choice is different depending on the language(s) I'm using. IDLE (despite its flaws) is my favorite for Python, but command-line/nano is a close second. For web development I love Atom (I'm using Atom right now to write this). When all else fails Notepad++ is extremely versatile. I'll also use Visual Studio if threatened.

My language of choice for Machine Learning is Python. Specifically I use a numerical computation module called Theano. I've used scikit-learn, PyBrain, Pylearn2, but I find none of them compare. Tensorflow is very powerful as well, but I prefer the fine-grianed control of Theano.

Finally I rely heavily on my background in mathematics to build and train models. I use it to approach machine learning algorithms in new ways and think about them beyond pure code. I refused to create a Neural Network in any language until I had derived the formula for Backpropagation by hand. Ultimately I believe a thorough understanding of math and code is required to be successful when creating complex models.
